# WizeKit Slack App Frontend

Frontend project for wizekit-slack-app app


## Development

Install dependencies

```
npm i
```

Start
```
npm run develop
```

# Wizekit slack app backend

The project is deploying on firebase at: https://wizekit-app-065aa1bc.web.app/
